# Stylesheets for AiiDAlab Widgets Base

This folder contains the following:

- `scss` - a folder for SCSS-format stylesheets to be compiled
- `.css` files  - compiled CSS stylesheets

It is recommended to add SCSS stylesheets in the SCSS folder for functionality and flexibility (see https://sass-lang.com/).

Compiling SCSS into CSS can be done using:

- The [compile-hero](https://marketplace.visualstudio.com/items?itemName=Wscats.eno) extension, if using VS Code
- The `sass` [CLI](https://sass-lang.com/install/)

All CSS stylesheets in the CSS folder are loaded on any import from the AiiDAlab widgets base package.
